Main Projects

  • [ASR+NLP]Multilingual Voice-Controlled Smart AC System for LG: This project enables the control of LG air conditioners using voice commands in multiple languages(English/Hindi/korean), enhancing user convenience with the ability to interact in English, Korean and Hindi. The system leverages AI-driven voice recognition as well as Intent and slot filling for seamless keyword extraction & operation, providing an intuitive and multilingual solution for smart home climate control.
  • [Gen AI]Autonomus Home Agent: This project focuses on developing an autonomous agent for home environments, designed to perform tasks such as monitoring, controlling, and optimizing household functions. Leveraging artificial intelligence and machine learning, the agent can learn from its surroundings and make decisions to enhance home automation, improve energy efficiency, and ensure safety. The system can integrate with smart devices to automate daily activities and provide real-time responses based on user preferences and environmental conditions.
  • [Time Series]Airbase Runway Visibility Forecasting system for Indian Air Force: This project focused on developing and deploying multivariate time series deep learning models to forecast runway visibility at Indian Air Force airbases for multiple future time horizons. Leveraging historical meteorological data such as temperature, humidity, wind speed, and precipitation, the model provided accurate visibility predictions. The solution enhanced the Indian Air Force's operational planning and mission execution in adverse weather conditions, contributing significantly to mission readiness and safety.
  • [CV]Face Recognition System: This project leverages a Syemis neural network architecture to accurately identify and verify individuals based on their facial features. By training the model on a diverse dataset of images, it learns to extract unique facial characteristics and achieve high accuracy in recognition tasks. The system showcases the effectiveness of deep learning techniques in real-world applications, such as security and user authentication.
  • [NLP]Automated Resume Screening Tool for Recruiters: This project involved the development of an automated resume screening tool to streamline the recruitment process. By leveraging natural language processing (NLP) techniques, the tool efficiently extracts relevant skills and experience from resumes and matches them with job descriptions, enabling recruiters to quickly identify qualified candidates and improve hiring efficiency.

GitHub Code Post

  • Gradient Calculation in Neural Network with Karpathy's Micrograd: In this Code post, we’ll explore how gradients are calculated in neural networks, based on Karpathy’s insightful lecture and his Micrograd library. Gradients play a crucial role in optimizing the performance of neural networks, and understanding how they work is key to grasping backpropagation and network training.
  • Bigram character level Language modeling: This code post explores the concept of Bigram Character-Level Language Modeling, which focuses on predicting the next character in a sequence based on the previous character. By utilizing bigrams, we can capture local dependencies in text data, making it a powerful approach for tasks such as text generation and completion.
  • Character Level Language Modeling Using MLP (Multi-Layer Perceptron): This code post presents a character-level language modeling approach utilizing a Multi-Layer Perceptron (MLP) to predict the next character in a sequence based on a given input. The model is trained on a dataset containing approximately 32,033 different names in the English language, enabling it to learn the patterns and structures inherent in character sequences.
  • Language Translation with Transformer: This Code post focuses on developing a language translator that converts English text into Italian using an attention-based Transformer architecture. The goal is to implement a robust translation model capable of understanding and generating accurate translations.
  • Character Level Language Modeling Using GPT: This Code Post focuses on implementing a character-level language model using the Generative Pre-trained Transformer (GPT) architecture. The objective is to generate coherent text by predicting the next character in a sequence based on previous characters, leveraging the powerful capabilities of GPT.
